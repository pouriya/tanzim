# tanzim-parse/examples

| File | Description |
|------|-------------|
| `parse_config.rs` | CLI: read a config file by path, auto-select parser by extension, print the value tree |
| `custom_parser.rs` | Custom `Parse` impl for `key=value` lines; shows auto-detection and error reporting |

`parse_config.rs` requires feature `full`. `custom_parser.rs` has no required features.
